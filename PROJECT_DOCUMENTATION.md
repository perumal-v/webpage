## 4. API Usage

This project uses **1 External API** to handle form submissions without a backend server.

### **Web3Forms**

- **Purpose**: Sends emails from the "Contact Us" form directly to your inbox.
- **Location**: Used in `src/Components/Email.jsx`.
- **Key**: The Access Key is hardcoded in the file.
- **Setup**:
  1.  Go to [Web3Forms](https://web3forms.com/).
  2.  Create an Access Key for your email address.
  3.  Replace the key in line 14 of `src/Components/Email.jsx`:
      ```javascript
      formData.append("access_key", "YOUR-NEW-ACCESS-KEY");
      ```

## 5. How to Run This Project (Local Development)

Follow these exact steps to run the website on your computer:

1.  **Install Node.js**: Ensure you have Node.js installed on your computer.
2.  **Open Terminal**: Navigate to the `frontend` folder.
3.  **Install Modules**:
    Run this command to download all unnecessary tools (React, Vite, etc.):
    ```bash
    npm install
    ```
4.  **Start Server**:
    Run this command to turn on the website:
    ```bash
    npm run dev
    ```
5.  **View Website**: Open your browser and go to `http://localhost:5173`.

## 6. How to Deploy (Go Live)

This is a **static website**, which makes it free and easy to host. You can use **Netlify**, **Vercel**, **GitHub Pages**, or standard hosting (Hostinger/GoDaddy).

### **Option A: Deploy to Netlify (Recommended - Easiest)**

1.  Create an account on [Netlify](https://www.netlify.com/).
2.  Drag and drop your `dist` folder onto their dashboard.
    - _Note_: To get the `dist` folder, run `npm run build` in your terminal first.
3.  Your site is now live!

### **Option B: Manual Deployment (cPanel / Hostinger)**

1.  **Build the Project**:
    In your terminal, run:
    ```bash
    npm run build
    ```
    This creates a new folder named `dist`. This folder contains the **final, optimized version** of your website.
2.  **Zip the Folder**:
    Compress the contents of the `dist` folder (Index.html, assets folder, etc.) into a ZIP file.
3.  **Upload**:
    Go to your hosting File Manager (public_html folder) and upload the ZIP.
4.  **Extract**:
    Unzip the file. Ensure `index.html` is in the main directory.
